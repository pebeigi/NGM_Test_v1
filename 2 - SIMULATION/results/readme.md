# results

Post-processing tools and outputs for simulation runs: analysis notebooks, plotting and safety-metric
scripts, baseline configuration snapshots, and batch/grid outputs.

## Analysis scripts

| File | Description |
|------|-------------|
| `safety_metrics.py` | Standalone **surrogate-safety** analysis on trajectory CSVs: V2V TTC (1-D on links, 2-D inside intersections), V2P/V2B predictive 2-D TTC, and PET via a spatial grid. Writes histograms/CDFs and summary CSVs to `processed_outputs/`. |
| `post_sim_plots.py` | Lightweight post-run plotting called from `GUI.py` after a trajectory CSV is saved. Modes: `trajectory_xy`, `time_space` (per-lane), `flow_density` (approximate q–ρ per lane). Entry point `run_post_sim_plots()`. |

## Notebooks

| File | Description |
|------|-------------|
| `Results_Processing.ipynb` | Post-processing for Case Studies 1–4: discovers run CSVs under `simulation_results_dir()`, plots logical-lane trajectories and EDIE-style flow–density curves, and computes TTC distributions. Writes `processed_outputs/analysis_summary.csv`. Uses [`ngm_paths.py`](../../ngm_paths.py). |

## Baseline snapshots (`*.pkl`)

`GUI.py` writes a full configuration snapshot on each run so batch scripts can re-run without the GUI:
`baseline_freeway.pkl`, `baseline_arterial.pkl`, `baseline_single_intersection.pkl`,
`baseline_tgsim.pkl`, and freeway sub-layout variants (`baseline_freeway_on_off.pkl`, …). The batch
runners (`../run_case_studies.py`, `../run_tgsim_ddm_grid.py`, `../run_tgsim_mobil_grid.py`) load these,
override the fleet mix / communication parameters, and re-run headless.

## Trajectory CSVs

| File | Description |
|------|-------------|
| `ptddm.csv` | Large saved trajectory run (TGSIM-style lanes `Main_*`, `NB_*`, `SB_*`). |
| `SW_AV.csv` | Freeway shockwave-case trajectory (AV fleet). |
| `test_sim.csv` | Example trajectory output (when present); the GUI writes to the user-chosen `Data_Folder/test_sim.csv`. |

> Large grid/trajectory CSVs are heavy; they are excluded from version control by the root
> [`.gitignore`](../../.gitignore) (`*.csv`).

## Subfolders

| Folder | Contents |
|--------|----------|
| `tgsim_ddm_grid/` | Output of `run_tgsim_ddm_grid.py`: TGSIM runs sweeping DDM lane-change levels (`L0_default` … `L4_extreme`) + `DDM_grid_manifest.csv`. |
| `tgsim_mobil_grid/` | Output of `run_tgsim_mobil_grid.py`: TGSIM runs with MOBIL presets (`L0_many_LC` … `L4_few_LC`) + `MOBIL_grid_manifest.csv`. |
| `case_runs/` | Per-scenario run CSVs from `run_case_studies.py` (`<label>/run_##.csv`). Git-ignored. |
| `processed_outputs/` | Figures and summary CSVs from the analysis notebooks/scripts. Git-ignored. |
| `test_sim_time_space_plots/`, `test_sim_flow_density_plots/` | Per-lane plots generated by `post_sim_plots.py`. |
